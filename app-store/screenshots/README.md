# App Store Screenshots — FocusLock

## Required sizes
- iPhone 6.9" (iPhone 16 Pro Max): 1320×2868 px
- iPhone 6.7" (iPhone 15 Pro Max): 1290×2796 px
- iPad 13" (required if supporting iPad): 2064×2752 px

## Screens to capture

1. **Home / Timer** — "Block distractions. Get in the zone."
   Timer set at 50 min, session about to start

2. **Active Session** — "Stay locked in. Your best work is happening."
   Timer counting down, minimal lock screen UI

3. **Session Complete** — "Session complete. You just did 90 minutes of deep work."
   Results screen showing time + streak

4. **History / Stats** — "Track your deep work hours. Watch yourself grow."
   Weekly chart, total hours, streak counter

5. **Widget** — "Start a focus session from your home screen."
   iPhone home screen showing FocusLock widget

## How to capture (Jabree — requires Apple Dev credentials)

1. Open Xcode → Product → Destination → iPhone 16 Pro Max simulator
2. Build and run FocusLock
3. Navigate to each screen listed above
4. File → Save Screenshot in Simulator (or Cmd+S)
5. Screenshots save to Desktop by default
6. Export at correct resolution for each device size

## Notes
- No device frames needed (Apple provides them in App Store Connect)
- Use light mode screenshots (higher conversion rate)
- Overlay headline text can be added in Canva or Figma
